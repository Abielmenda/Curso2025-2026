Application - Museums of Madrid Explorer

General Description

Museums of Madrid Explorer is a web application guiding users through Madrid’s iconic museums like the Prado and Reina Sofía. It offers interactive maps with general info of the museum.

Objectives

  • Display an interactive map of Madrid’s museums for easy navigation.
  • Visualize curated highlights and exhibit details for each museum.
  • Provide real-time updates on exhibitions, ticket prices, and hours.

Requirements

Functional Requirements

  • FR1: Display an interactive map with museum locations and key details.
  • FR2: Filter museums by type, proximity, or exhibition themes.
  • FR3: Allow offline access to museum guides and exportable itineraries.

Non-functional Requirements

  • NFR1: Ensure a responsive and accessible interface compatible with desktop devices.
  • NFR2: Support multilingual content for at least English, Spanish, and French.
  • NFR3: Implement automatic updates for museum data every week.

Mockup

  • Screen 1: Interactive map of Madrid with pinned museum locations, color-coded by type (e.g., art, history).
  • Screen 2: Gallery view showcasing key exhibits with images and brief descriptions, filterable by theme or museum.
  • Screen 3: Itinerary planner displaying suggested routes, schedules, and notifications for upcoming exhibitions.
